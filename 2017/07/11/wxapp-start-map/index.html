小程序入门－地图 定位功能开发

发表于 2017-07-11 | 分类于 小程序 | | 阅读次数

对于我们普通开发人员来说，小程序就是微信家开发的一套框架，那么这个框是怎么样的，怎么用它开发一个小程序应用，这应该是我们需要明白理解的。

简单理解下这个框架，它其实就是定义了一些自己的标记语言，样式语言以及页面路由，数据绑定等一整套的规则，然后将 我们按照它的这些规则开发的代码最终生成了一个应用 小程序。

通过下载官方开发者工具创建一个quick start 项目可以看到，小程序整体结构由两部分组成，第一部分是三个描述整个程序的app文件；第二部分是描述每个页面的pages文件部分。（utils中存放的是一些公用模块）

app.js 通过App()函数来管理整个小程序的生命周期，全局数据方法等，app.json是全局配置文件，app.wxss是全局样式文件，pages部分每一个页面由相同路径及文件名的 .js ,.json, .wxml, .wxss四个文件组成

总结：

1、开发中如果手机预览时，发现背景图片不显示，可以通过base64，网络图片来解决，
2、map组件的层级 即z-index值在整个页面中是最高的，示例中处理方式是当显示城市列表时将map给隐藏了。
3、对于类似小程序这样数据双向绑定的框架，开发中需要改变的一个思路就是 不再是之前jquery那样对dom的操作，而是转变为对数据data的操作。
