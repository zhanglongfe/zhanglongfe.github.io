es6 class 类

发表于 2017-04-05 | 分类于 javascript | | 阅读次数

以前js使用构造函数的方式定义生成一个对象，而在es6中引入了class 类的概念来生成对象

＊用于定义类的方法不需要关键字function，可以直接写方法名

＊所有class定义的类内部会有一个construnctor构造方法，相当于传统方法定义的构造函数；用来构造对象并返回；如果没有显示的定义，内部会自动隐式生成一个空的construnctor(){}

class 继承

＊ class痛过关键字extends实现继承父类所有的属性 方法

＊ 子类的construnctor 方法中必须调用super,super表示的是父类的构造函数。这是因为子类没有自己的this对象，它的this对象是继承父类的this对象，然后用自己的构造函数来加工

＊ super表示的是父类的构造函数,所以也是父类本身，所以可以调用它prototype的方法：super.toString()

class的继承机制

es5中通过原型链__proto__实现继承，而es6 class的继承是咋样的呢？

class作为构造函数的语法糖，它同时拥有prototype和__proto__两个属性，因此它有两条继承链

＊ 子类B作为对象它拥有__proto__属性，指向它的构造函数，由于它是继承自A，所以它指向A的构造函数，也就是A 本身

＊子类B作为构造函数，它的prototype的__proto__指向了父类的prototype，从而来继承父类的方法，因为class的方法全部来自它的prototype对象。
